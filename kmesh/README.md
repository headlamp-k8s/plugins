# KMesh

KMesh Headlamp plugin providing visibility into KMesh Waypoint resources (Gateway API Gateways with `gatewayClassName: kmesh-waypoint`).

## Features
- List Waypoints and open details in a split-right panel.
- View key fields (Gateway Class, Image, Status) in the details view.
